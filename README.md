# IMDB.drama Clustering Project

Projektni zadatak iz istraživanja podataka.

## Tema
Klasterovanje skupa podataka IMDB.drama sa OpenML.

## Ciljevi
- preprocesiranje podataka
- primena minimum 5 algoritama klasterovanja
- poređenje rezultata
- vizualizacija u 2D/3D
- modeli sa svim atributima i redukovanim skupovima atributa
- čuvanje početnih i preprocesiranih podataka i konstruisanih modela
